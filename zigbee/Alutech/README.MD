Улучшенная версия шаблона к SprutHub для zigbee-модуля управления рольставнями Alutech.
https://sprut.ai/catalog/item/id/3730

Улучшения в сравнении со встроенным шаблоном:
- не сбрасываются настройки при спаривании со Spruthub
- в настройки устройства добавлена возможность калибровки конечных позиций
- PositionState (целевое состояние) привязано к атрибутам Zigbee напрямую
- пересчет процентов открытия из стандарта Zigbee в стандарт Spruthub/Homekit теперь в шаблоне - не нужно для каждого устройства настраивать по два раза инверсию характеристик
- добавлена автоматическая привязка логики "Целевая позиция зависит от текущей" - для красивого отображения статуса
- выведены информационные поля
